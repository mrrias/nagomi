# Nagomi

Welcome to the official website repository for **Nagomi** — a friendly community centered around **mental health, anime, and LGBTQ+ pride**.

## About

Nagomi is a welcoming and inclusive space where members can:

- Connect with supportive people
- Discuss mental health and wellbeing
- Explore LGBTQ+ identities and experiences
- Chat about anime, manga, and related media
- Participate in community events and activities

This website serves as an information hub for the community and provides visitors with an overview of what Nagomi has to offer.

## Features

- Clean and modern design
- Light and dark mode support
- Mobile-friendly layout
- Dedicated pages for community topics
- Easy navigation
- Fast and lightweight
