About

Shiro is Nagomi's personal helper, created to keep the server active, welcoming, and fun. Whether it's bringing members together through interactive commands, adding a little entertainment to chat or helping with everyday server tasks. Shiro is always around to make the community more enjoyable for everyone.

Getting Started

To get started use the slash command /help and ask for the list of commands to be sent to your DMs.
If you are stuck at all or having any struggle please feel free to ping @mio.miyuki with any questions. Also feel free to leave feedback on how Shiro can better help improve out community.
